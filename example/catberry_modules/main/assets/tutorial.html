Where am I?

This is a demo application of catberry framework. It is a simple chat service with minimal security but with push notifications about new messages.

The main goal of this demo is to show you how to develop modules for Catberry framework and application in common. Please go to sources of this application and read some comments about that.

Also you can find "About" page with details about Catberry framework.
